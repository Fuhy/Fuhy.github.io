Usage

position is where the string begins, and offset defines how long it lasts. If offset is not seted, it will be seted as the length of the strings. Hope this string is end with ‘\0’, or here comes SEGMENTATION FAULTS.
color is one of the string {“black”, “red”, “green”, “yellow”, “blue”, “purple”, “azure”, “white”}. Or you can make it a serial number like “1” or “31”, which means “red”

The function colorPrint returns a pointer which points to a new array.
That means you should free the allocated memory after uses by using the fuction free_colorPrint.

实现

控制字串"\033[00;00;00m"可以控制字的背景、颜色、效果。其中30～37代表了8种可选颜色"black", "red", "green", "yellow", "blue", "purple", "azure", "white"(我随便翻译的

而字串"\033[0m"将所有效果清空。

函数colorPrint只是简单地封装了构建包含控制字串的新字符串的过程。其使用malloc分配内存空间给新的字串并将构建好的字符串首部指针返回。

  • 若offset没有指定，则使用库函数strlen（position）获取长度。但这里显然很天真地期待了这是一个合法的字符串。若是字符串没有以’\0’结尾，则可能会访问非法内存。

函数free_colorPrint释放所有之前为新字符串分配的内存。指向新构建的字符串指针会被存储在设为static的外部变量vector中。此函数只是简单的释放vector中的所有指针。
